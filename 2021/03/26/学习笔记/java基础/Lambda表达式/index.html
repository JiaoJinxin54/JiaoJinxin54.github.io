Lambda表达式

发表于 2021-03-26 | 分类于 学习笔记 ， java

概念

简化了对函数式接口的实现。

语法格式

(参数类型 参数)->{主体}

重要特征

  • 可选声明类型：参数的类型可以不做声明
  • 可选参数圆括号：参数为1个时，可不写圆括号
  • 可选主体大括号：主体只有一个表达式时，可不写大括号
  • 可选返回关键字：主题只有一个表达式，且表达式结果为返回值时，可不写返回关键字return

优点

  • 代码更简洁紧凑

注意

  • lambda表达式内如果引用基本类型的外部变量，则该变量只能是被final修饰，由于被final修是，故不可更改。
  • lambda表达式中调用的外部变量实际不是不可变的，只是引用不可变，所以如果要使用或者修改外部基本类型变量，则可以通过封装的方式解决，即将基本类型封装到引用类型中，调用方法进行改变该引用类型的属性的值。
