链式编程

发表于 2021-03-26 | 分类于 学习笔记 ， java

概念

为了提高可读性的一种代码编写方式，即当方法本身无需返回任何信息时，可以将对象本身返回，达到链式编程的效果，非必须使用，但不可为了使用而滥用。

优点

  • 代码简洁
  • 可读性强
  • 代码主体逻辑更清晰

缺点

  • 不太利于代码调试
  • 对开发人员的业务能力要求高
