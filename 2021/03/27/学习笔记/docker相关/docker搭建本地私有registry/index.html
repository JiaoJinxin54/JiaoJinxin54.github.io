启动镜像

  • -it: 后台允许
  • –name docker-registry: 设置容器名字为docker-registry
  • –restart=always: 自动重启
  • –privileged=true: centos7中的安全模块selinux把权限禁止了，加上这行是给容器增加执行权限
  • -p 5000:5000: 端口映射
  • -v /opt/docker/registry:/var/lib/registry: 将/var/lib/registry挂在到宿主机的/opt/docker/registry目录
