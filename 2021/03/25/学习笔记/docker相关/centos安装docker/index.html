centos安装docker

发表于 2021-03-25 | 分类于 学习笔记 ， docker

本文主要为了记录在腾讯云上安装docker的步骤，方便后续遗忘后进行查看。

1. 查询已安装的旧版docker

2. 卸载已安装的旧版docker

卸载上一步中查询到的已安装的docker，卸载完成后再次执行”yum list installed | grep docker“进行查询，若什么都查不到，则旧版本已卸载完成。

3. 安装docker

4. 查看是否安装成功

若可以正常显示docker版本，即表示docker已安装完成。

5. 更换国内镜像源

修改daemon.json，按i进入编辑模式，将以下内容复制进去，按esc退出编辑模式，输入 :wq退出编辑并保存。

上述地址只需要有一个即可，三个地址分别为：

  • Docker中国区官方镜像：https://registry.docker-cn.com
  • 网易：http://hub-mirror.c.163.com
  • 中国科技大学：https://docker.mirrors.ustc.edu.cn

6. 启动docker

7. 查看是否启动，且镜像源是否更换成功

若启动正常则会显示很多信息，其中Registry Mirrors: 后的内容即为镜像源，若与daemon.json中修改内容一致，则表示已切换成功，至此docker已安装完成，且已更换国内镜像源，并成功启动，接下来就可以进行docker相关操作了。
